# Arduino Traffic Light (WOKWI)

Simple Arduino project that simulates a traffic light using three LEDs.

## Components

- Arduino Uno
- Resistor 220Ω (3)
- LED (3)

## Features

- Simulates a traffic light using three LEDs
- Automatic light sequence (green → orange → red)
